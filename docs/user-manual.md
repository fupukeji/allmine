# 用户使用手册

## 概述

时间价值计算器是一款帮助您管理和追踪预付费资产价值的工具。通过记录您的年付会员、健身卡、软件订阅等预付费项目，系统会自动计算已消耗的成本和剩余价值，让您更清楚地了解自己的投资回报。

## 核心概念

### 价值计算原理
系统基于"时间消耗=价值消耗"的原则进行计算：

- **单位时间成本** = 总支付金额 ÷ 有效时长（天数）
- **已消耗成本** = 单位时间成本 × 已使用时长
- **剩余价值** = 总支付金额 - 已消耗成本
- **消耗进度** = 已使用时长 ÷ 有效时长 × 100%

### 项目状态
- **未开始**: 当前时间早于开始计算时间
- **消耗中**: 项目正在使用期间
- **已过期**: 当前时间晚于结束时间

## 详细使用指南

### 1. 账号管理

#### 注册新账号
1. 访问系统首页
2. 点击"立即注册"
3. 填写用户名（3-20个字符）
4. 填写邮箱地址
5. 设置密码（至少6位）
6. 确认密码
7. 点击"注册"按钮

#### 登录系统
1. 在登录页面输入用户名或邮箱
2. 输入密码
3. 点击"登录"按钮

*默认管理员账号：admin / admin123*

### 2. 分类管理

分类用于对项目进行归类，便于管理和统计。

#### 创建分类
1. 点击侧边栏"分类管理"
2. 点击"添加分类"按钮
3. 输入分类名称（不超过50个字符）
4. 选择分类颜色：
   - 点击预设颜色块快速选择
   - 或输入自定义颜色代码（如 #FF5733）
5. 输入图标名称（如 folder, star, trophy）
6. 点击"确定"保存

#### 编辑分类
1. 在分类列表中找到要编辑的分类
2. 点击"编辑"按钮
3. 修改分类信息
4. 点击"确定"保存

#### 删除分类
1. 在分类列表中找到要删除的分类
2. 点击"删除"按钮
3. 确认删除操作

*注意：只有没有关联项目的分类才能被删除*

### 3. 项目管理

项目是系统的核心，代表您的各种预付费资产。

#### 添加项目
1. 点击侧边栏"项目管理"
2. 点击"添加项目"按钮
3. 填写项目信息：

**基本信息**
- **项目名称**: 描述性名称，如"某某健身房年卡"
- **所属分类**: 从下拉列表选择分类
- **总支付金额**: 您实际支付的金额

**时间设置**
- **开始计算时间**: 项目开始生效的时间（如会员激活时间）
- **结束时间**: 项目到期时间
- **购买时间**: 实际付款时间（可选，仅作记录）

**备注信息**
- **购买目的**: 购买这个项目的原因或用途（可选）

4. 点击"确定"创建项目

#### 查看项目详情
1. 在项目列表中找到目标项目
2. 点击"详情"按钮
3. 查看完整的计算信息和项目详情

#### 编辑项目
1. 在项目列表中找到要编辑的项目
2. 点击"编辑"按钮
3. 修改项目信息
4. 点击"确定"保存

#### 删除项目
1. 在项目列表中找到要删除的项目
2. 点击"删除"按钮
3. 确认删除操作

#### 批量删除
1. 勾选多个要删除的项目
2. 点击"批量删除"按钮
3. 确认删除操作

### 4. 数据筛选和排序

#### 筛选功能
- **按分类筛选**: 选择特定分类查看该分类下的项目
- **按状态筛选**: 选择项目状态（未开始/消耗中/已过期）

#### 排序功能
项目列表支持多种排序方式：
- 创建时间（默认）
- 剩余价值
- 消耗进度
- 总金额

### 5. 仪表盘统计

仪表盘提供了全局的统计信息：

#### 统计卡片
- **项目总数**: 您添加的所有项目数量
- **总投入金额**: 所有项目的支付金额总和
- **已消耗成本**: 所有项目已消耗成本的总和
- **剩余价值**: 所有项目剩余价值的总和

#### 项目状态分布
显示不同状态项目的数量统计

#### 最近项目
显示最近创建的5个项目及其基本信息

## 使用技巧

### 1. 合理设置时间
- **开始时间**：应设为实际可以开始使用的时间，而非购买时间
- **结束时间**：准确设置到期时间，便于提前规划续费

### 2. 分类管理
- 建议创建符合个人习惯的分类体系
- 常见分类：运动健身、技术工具、学习提升、生活服务、娱乐休闲

### 3. 定期检查
- 建议每周或每月查看一次仪表盘
- 重点关注即将到期的项目
- 评估低使用率的项目是否需要调整

### 4. 价值评估
- 通过"已消耗成本"了解已经投入的金额
- 通过"剩余价值"规划未来使用
- 通过"消耗进度"评估使用效率

## 数据安全

### 备份建议
- 系统数据存储在本地数据库
- 建议定期导出重要项目信息
- 可以截图保存统计数据

### 隐私保护
- 项目信息仅存储在您的本地环境
- 不会上传到外部服务器
- 请妥善保管登录密码

## 常见问题

**Q: 为什么我的项目显示"未开始"状态？**
A: 当前时间早于您设置的"开始计算时间"，系统认为项目还未生效。

**Q: 如何修改已经创建的项目？**
A: 在项目列表中点击"编辑"按钮，可以修改除ID外的所有信息。

**Q: 删除分类后，相关项目会怎样？**
A: 系统不允许删除有关联项目的分类，需要先将项目转移到其他分类或删除项目。

**Q: 计算结果不准确怎么办？**
A: 请检查项目的时间设置是否正确，特别是开始时间和结束时间。

**Q: 可以批量导入项目吗？**
A: 当前版本需要逐个添加项目，未来版本会考虑添加批量导入功能。
